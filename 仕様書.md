# AIチャットボット 仕様書

## 概要

最低限のチャット機能を備えたAIチャットボットアプリケーションをNext.jsで実装する。
ユーザはメインプロンプト、役割、スキルを入力し、OpenAIのAPIを通じてチャットを行うことができる。チャット履歴はサイドバーに表示され、過去の会話にアクセスできる。
dbは使用せず、サンプルデータを用いてチャット履歴を管理する。
まずはAPIにアクセスせず、固定の文字列を返すことで、フロントエンドのUIとUXを優先的に実装する。

## 実装範囲

- チャット画面
  - チャット機能
    - メインプロンプトの入力
    - ユーザの役割、スキルの入力
    - OpenAIのAPIへプロンプトを送信（後ほど実装）
    - OpenAIのAPIからのレスポンスを受信（後ほど実装）
    - レスポンスを表示
  - (サイドバー)チャット履歴
    - チャット履歴一覧の取得
    - チャット履歴の表示
    - チャット履歴の選択

## 使用技術

- フロントエンド・バックエンド共通
  - Next.js
  - shadcn/ui（UIコンポーネントライブラリ）
  - Tailwind CSS（スタイリング。カラーテーマ定義）
- OpenAI API（チャット機能のバックエンド）

## 型定義（Types.ts）

### 概要

export interfaceを使用して、各データ構造の型を定義する。

```typescript
export interface ChatHistorySummaryObject {
  chatId: string;
  title: string;
  createdAt: string;
  updatedAt: string;
}
```

### 型情報

- チャット履歴一覧（ChatHistorySummaryObject）
  - chatId: string (チャットの一意な識別子)
  - title: string (チャットのタイトル。最初のユーザーメッセージの一部を使用)
  - createdAt: string (チャットの作成時間)
  - updatedAt: string (チャットの最終更新時間)
- チャット履歴詳細（ChatHistoryObject）
  - chatId: string (メッセージの一意な識別子。時間に依存したUUID)
  - messages: Message[] (メッセージの配列)
  - createdAt: string (チャットの作成時間)
  - updatedAt: string (チャットの最終更新時間)
- メッセージ (MessageObject)
  - role: 'user' | 'assistant' (メッセージの役割)
  - content: string (メッセージの内容)
  - timestamp: string (メッセージのタイムスタンプ)
- 入力プロンプト (InputPromptObject)
  - mainPrompt: string (メインプロンプト)
  - userRole: string (ユーザの役割)
  - userSkills: string (ユーザのスキル)
